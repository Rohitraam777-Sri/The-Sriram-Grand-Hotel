 # The-Sriram-Grand-Hotel

The-Sriram-Grand-Hotel is a simple front-end hotel website built using HTML, CSS, and JavaScript. It displays different food menu sections such as Breakfast, Lunch, Dinner, and Snacks with a clean and interactive user interface.

## Features
- Hotel menu website design
- Separate pages for food items
- Clean and simple UI
- Easy navigation between menu sections
- Responsive layout

## Technologies Used
- HTML
- CSS
- JavaScript
